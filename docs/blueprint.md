# **App Name**: Visitor Management System

## Core Features:

- Visitor Check-in Form: A single form on the landing page allowing visitors to enter their name, company, and purpose of visit.

## Style Guidelines:

- The visual design emphasizes professionalism and clarity through a light color scheme. The primary color is a muted, trustworthy blue (#587CAF), chosen to evoke a sense of security and efficiency. It is versatile enough to be used for key interactive elements without being overpowering.
